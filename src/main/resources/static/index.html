FairDraw

Home · Host a room · Join a draw

FairDraw shuffles the numbers 1–50 once per room, with no repeats. Hosts create a room and share codes; each participant uses their own code to draw and follow the pool.

Host a room Set how many people join, get a room code and one secret code per person. Join a draw Enter the code you were given, draw numbers, and see the shared pool and log.
